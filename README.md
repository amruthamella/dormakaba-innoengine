# AI-Powered Innovation Blueprint Generator

An intelligent web application developed during my internship at Dormakaba to assist research and development teams in generating structured innovation blueprints, evaluating ideas, and streamlining the innovation planning process.

---

## Overview

The AI-Powered Innovation Blueprint Generator helps organizations transform raw ideas into actionable innovation plans through automated analysis and structured documentation.

The platform enables users to:

- Generate innovation blueprints from project concepts
- Organize research and development ideas
- Structure innovation workflows
- Improve documentation efficiency
- Support decision-making through AI-assisted recommendations

## Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/dormakaba-innoengine.git
```

### Navigate to Project Directory

```bash
cd dormakaba-innoengine
```

### Install Dependencies

Frontend:

```bash
cd frontend
npm install
```

Backend:

```bash
cd backend
npm install
```

### Start Application

Frontend:

```bash
npm start
```

Backend:

```bash
npm run dev
```
